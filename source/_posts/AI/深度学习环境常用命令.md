1. 创建虚拟环境
    ~~~
    conda create –n name python=3.7
    ~~~
2. 激活环境
    ~~~
    activate name
    ~~~
3. 安装虚拟环境所需配置
   - 有`requirements.txt`
        ~~~
        pip install -r requirements.txt
        ~~~
   - 无`requirements.txt`
        ~~~
        pip install name==version
        ~~~
4. 查看现有虚拟环境
    ~~~
    conda env list
    ~~~       
5. 删除虚拟环境
    ~~~
    conda remove -n  name --all
    ~~~
