1. 创建环境
    ~~~
    conda create –n name python=3.7
    ~~~
2. 激活环境
    ~~~
    activate name
    ~~~
3. 安装环境所需配置
   - 有`requirements.txt`
        ~~~
        pip install -r requirements.txt
        ~~~
   - 无`requirements.txt`
        ~~~
        pip install name==version
        ~~~
